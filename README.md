Solutions for https://adventofcode.com/

This repository intentionally omits puzzle texts and inputs because AoC author request not to share them. [Link to AoC about page.](https://adventofcode.com/2024/about#:~:text=Can%20I%20copy/redistribute%20part%20of%20Advent%20of%20Code%3F)

Intended way to run code for some puzzle is to create folder `advent-of-code-input` next to folder of this repository with folder of year and file with name `D{two_digin_day_number}.txt` (Example `advent-of-code-input/2023/D05.txt`), inside which belongs the input retrieved from Advent of Code website.
